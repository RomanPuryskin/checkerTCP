# checkerTCP
Запуск:

В файле example.env представлены значения переменных окружения по умолчанию при запуске. Если хотим запустить со своими значениями, создаем в корне файл .env, копируем содержимое в него файла example.env и подставляем свои значения.

1) Запускаем сервер в docker:
   docker-compose up -d

2) Запускаем клиент (CLI)
   - командой ./cmd/client/main.go
   - или в корне проекта выполнить команду make build  -> запускаем бинарник из папки ./bin
